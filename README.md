# 新建txt

一个可以拖到 Finder 工具栏的 macOS 小应用。点击后会在当前 Finder 窗口所在文件夹中新建空白 `.txt` 文件。

## 下载

从 GitHub Release 下载 `新建txt.app.zip`，解压后得到 `新建txt.app`。

## 使用方式

1. 下载或复制 `新建txt.app` 到 `~/Applications`。
2. 打开 Finder。
3. 按住 `Command`，把 `新建txt.app` 拖到 Finder 窗口顶部工具栏。
4. 点击工具栏里的图标，即可在当前文件夹中新建文本文件。

第一次运行时，macOS 可能会请求允许该应用控制 Finder。请选择允许。

## 文件命名

应用会创建：

```text
新建文本文件.txt
```

如果同名文件已存在，会自动创建：

```text
新建文本文件 2.txt
新建文本文件 3.txt
...
```

## 常见问题

### 提示“无法验证开发者”，打不开怎么办？

这是因为该应用没有经过 Apple Developer ID 签名和 notarize。请在 Finder 中右键点击 `新建txt.app`，选择“打开”，再在弹窗中确认打开。

如果仍然无法打开，可以到“系统设置” → “隐私与安全性”中允许打开该应用。

### 第一次运行时提示控制 Finder，要允许吗？

需要允许。应用必须访问 Finder 当前窗口，才能知道应该在哪个文件夹中新建文本文件。

如果误点了拒绝，可以到“系统设置” → “隐私与安全性” → “自动化”，找到 `新建txt`，允许它控制 Finder。

### 为什么拖不到 Finder 工具栏？

需要按住 `Command` 再拖动 `新建txt.app` 到 Finder 窗口顶部工具栏。普通拖动不会添加到工具栏。
